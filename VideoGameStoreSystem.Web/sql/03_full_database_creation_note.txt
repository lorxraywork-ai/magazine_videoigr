-- Полный SQL-скрипт создания БД для проекта
-- «Информационная система магазина видеоигр»
--
-- Что делает этот скрипт:
-- 1. Создает БД [magazine_videoigr], если ее еще нет.
-- 2. Создает служебную таблицу [__EFMigrationsHistory].
-- 3. Создает все таблицы, внешние ключи и индексы в финальных именах схемы проекта.
-- 4. Добавляет базовые роли приложения.
-- 5. Добавляет записи о миграциях, чтобы ASP.NET Core приложение не пыталось
--    повторно создать таблицы при вызове Database.Migrate().
--
-- Важно:
-- После выполнения этого скрипта можно запускать приложение.
-- Демонстрационные пользователи, товары, клиенты, поставки, продажи и списания
-- будут автоматически добавлены приложением при первом старте,
-- если соответствующие таблицы пока пусты.

USE [master];
GO

BEGIN
    CREATE DATABASE [magazine_videoigr];
END;
